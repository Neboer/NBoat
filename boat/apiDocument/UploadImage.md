# 上传图片

向nopiser上传一张图片，并获取图片的id，可以作为图片的索引写入博客数据中。

**URL** : `/api/nopiser/picture`

**方法** : `POST`

**请求体**：

请求体是一个form形式的数据。form中包含一个`image`字段，其值是一个type为file的input。
```
image:input(file)
```

**身份等级** : 游客

## 成功

**状态码** : `200 OK`

## 失败

**状态码** : `400`

当上传内容的MIME值不是图片，或者数据格式不正确、尺寸过大等时返回。这两个响应均不带响应体。
